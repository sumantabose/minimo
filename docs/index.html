This is the hugo source files repository for my website https://sumantabose.me/, powered by Hugo and Minimo.

If you'd like to replicate my website for your personal use, follow the 3 simple steps below.

  1. Download the master branch archive of the source repository from https://github.com/sumantabose/source/archive/master.zip
  2. hugo server to render the website locally at http://localhost:1313/
  3. hugo to build the website in the directory public/

To learn more about Hugo read the Hugo Documentation.
